# Blog Posts

Add your markdown files in this directory.

Each file should have a frontmatter section like this:

```markdown
---
title: "Your Post Title"
date: "YYYY-MM-DD"
description: "A short description of your post."
---
```

The filename (e.g., `my-post.md`) will be used as the URL slug (e.g., `/blog/my-post`).
